Sistema operativo:
Es el software principal que administra los recursos del computador. Permite ejecutar programas y gestionar archivos, memoria y dispositivos.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas, las respuestas y el comportamiento de la aplicación.

Puerto:
Es un punto de comunicación utilizado por el servidor para recibir solicitudes. En esta actividad se utiliza el puerto 3000.

Logs:
Son registros que almacenan información sobre la ejecución del sistema. Ayudan a identificar errores, monitorear actividad y realizar mantenimiento.
