El login es el proceso mediante el cual un usuario ingresa a una aplicación utilizando sus credenciales para verificar su identidad. Esto permite que cada persona acceda únicamente a las funciones que le corresponden. Algunas rutas deben estar protegidas porque contienen información o acciones que solo pueden realizar usuarios autorizados. De esta forma se evita el acceso no autorizado y se protege la información de la plataforma. No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, correos electrónicos, tokens, claves de API ni otros datos sensibles.
